

# Arus

Arus adalah sistem pencatatan keuangan harian sederhana untuk PT/Yayasan.

Fokus utama:
- Catat uang masuk
- Catat uang keluar
- Rekap otomatis
- Mobile-first
- Berbasis Profil Lembaga → Unit/Program → Kegiatan → Transaksi

Tech stack:
- CodeIgniter 4
- Tailwind CSS / SCSS
- MySQL / MariaDB
- WebView APK membuka website

Menu utama hanya:
1. Beranda
2. Catat
3. Rekap

Project ini bukan ERP, bukan aplikasi akuntansi penuh, dan bukan sistem pajak lengkap.
